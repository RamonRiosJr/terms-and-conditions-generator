# Terms and Conditions & Privacy Policy Generator

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A simple, single-file tool to help website owners generate basic **Terms and Conditions** and **Privacy Policy** drafts. Built with HTML, Vanilla JavaScript, and Tailwind CSS.

[**View Live Demo**](https://ramonriosjr.github.io/terms-and-conditions-generator/)

---

## âœ¨ Features

- **Dual Generators:** Create both *Terms & Conditions* and *Privacy Policies* from a single interface.
- **Tabbed Interface:** Easily switch between the two document types.
- **Comprehensive Sections:**
  - **Terms:** User Accounts, Acceptable Use Policy, User Content, Payments, Termination, Liability, and more.
  - **Privacy:** Data Collection, Cookies, Analytics, Ads, GDPR, and CCPA compliance.
- **Multiple Output Formats:**
  - **Plain Text:** For easy reading and copying.
  - **HTML:** Ready-to-paste code with semantic tags.
  - **Markdown:** Perfect for GitHub READMEs or documentation.
- **Auto-Save:** Your progress is automatically saved to your browser's local storage, so you never lose your work.
- **Zero Dependencies:** Fully self-contained in one `index.html` file (Tailwind CSS loaded via CDN).

## ğŸš€ How to Use

1. **Download:** Get the `index.html` file from this repository.
2. **Open:** Open the file in any modern web browser (Chrome, Firefox, Safari, Edge).
3. **Select Type:** Use the tabs at the top to choose between "Terms & Conditions" or "Privacy Policy".
4. **Fill Form:** Complete the fields with your website's details and select the clauses you need.
5. **Choose Format:** Select your desired output format (Text, HTML, or Markdown).
6. **Generate:** Click the **Generate Terms** button.
7. **Copy:** Click **Copy to Clipboard** and paste the result into your website or document.
